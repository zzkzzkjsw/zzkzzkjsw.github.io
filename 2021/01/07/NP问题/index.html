P and NP problem

1.1 P and NP

P: problem that can be solved in poly-time;

NP: problem that can be verified in poly-time;

1.2 NPC AND NP-Hard

NP-Hard: if problem in NP-Hard can be solved, then all of the NP problem can be solved.

NPC: if the problem is both a NP-Hard problem and a NP problem

1.3 Reduction

if p1 ——-poly-time——-> p2,
we say that p1 can be reduced to p2 , denoted by p1<=p p2,
and if p2 is solved, then p1 is solved

note: there are two steps to prove that p1->p2

  1. prove that p1 can be transformed to p2 in poly-time
  2. prove that p1 and p2 are equivlent

1.4 how to find the first NPC Promlem

we should prove that all of the NP problem can be reduced to NPC0 problem,

this is a very difficult task, but some NBer proved 3-SAT problem is NPC0

1.5 how to find if the problem is NPC

we can prove that all of the NP problem can be reduced to the problem,

but we do not need to do that,
we just need to prove that NPC1 can be reduced to NPC0,and NPC1 is NP problem
