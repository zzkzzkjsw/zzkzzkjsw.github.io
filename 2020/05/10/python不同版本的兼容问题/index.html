python不同版本的兼容问题

让python2.7，python3.6，python3.8其乐融融

第一步

安装python2.7，python3.6，python3.8,安装时已经有pip,将文件中的python.exe和pythonw.exe分别改为python2.7.exe和pythonw2.7.exe``python3.6.exe和pythonw3.6.exe``python3.8.exe和pythonw3.8.exe并设置好环境变量

第二步

在cmd中分别运行
python3.8 -m pip install --upgrade pip --force-reinstall
python3.6 -m pip install --upgrade pip --force-reinstall
python2.7 -m pip install --upgrade pip --force-reinstall

大功告成!

注意：
1.要区分这三个pip，要使用pip2.7``pip3.6``pip3.8分别取代pip
2.要区分这三个python，要使用python2.7``python3.6``python3.8分别取代python
===
备注：
1.python -m的意思是将库中的python模块用作脚本去运行。
主要区别在于sys.path不同
直接运行会将该脚本所在目录添加至sys.path
当做模块启动则会将当前运行命令的路径添加至sys.path
